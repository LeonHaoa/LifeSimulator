# Workflow Reference

## Recommended Order

1. Freeze project profile.
2. Run structural rollout.
3. Run polishing.
4. Run release validation.

## Source Locale Policy

1. `en` is the only source locale.
2. `zh-Hant` is the secondary priority locale.
3. Other locales always follow `en`.

## Module Priority

1. `Onboarding`
2. `Subscription / Paywall`
3. `Today / Greeting / Notification`
4. `Discover`
5. `Theme / Settings`
6. `Yearly / Daily / Insight`

## Content Priority

1. high-visible UI copy
2. repeated templates
3. long-form content
4. concept-heavy terminology
